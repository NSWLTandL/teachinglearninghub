# Nishkam Teaching & Learning Hub (flat build)

Static site for GitHub Pages. ALL files live in ONE folder — no subfolders.
This makes browser uploads reliable (drag all files into the repo; nothing nests).

Homepage: index.html (the Hub).
Other landing pages: programme-index.html, external-index.html, foundations.html, tl-programme.html.

## Upload (GitHub web)
1. In your repo: Add file -> Upload files.
2. Select ALL the .html files + README.md + .nojekyll and drop them in.
3. Commit. Settings -> Pages should already be on (main / root).
4. Site: https://nswltandl.github.io/teachinglearninghub/
